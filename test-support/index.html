JavaScript Toolkit Testing

Test Support

Support libraries for writing and running tests.

Blanket

blanket.js is a simple code coverage library for javascript. Designed to be easy to install and use, for both browser and nodejs.

Links

  • http://blanketjs.org/

Chai

Chai is an assertion library which allows different testing syntax.

Links

  • http://chaijs.com/

Istanbul

Yet another JS code coverage tool that computes statement, line, function and branch coverage with module loader hooks to transparently add coverage when running tests.

Links

  • https://github.com/gotwarlost/istanbul

Phantom JS

Phantom JS is a headless webkit browser with scripting capabilities.

Links

  • http://phantomjs.org/

Sinon

Sinon is a powerfull mocking library with test spies, stubs and mocks

Links

  • http://sinonjs.org/

Zombie JS

Zombie JS is a simulated browser environment to interact with web pages.

Links

  • http://zombie.labnotes.org/
