Crossroads.js is a library for routing URLs on the client side, directing users to specific views under single-page applications.

Director

Director is a routing library for the browser, that helps you decide which code to run accordig to the URL.

Links

localForage

Offline storage, improved. Wraps IndexedDB, WebSQL, or localStorage using a simple but powerful API.

Links

Minified

Minified is a light-weight utility library for DOM-manipulation, date & time, events, animations and much others under a single API.

Links

  • http://minifiedjs.com/

Page.js

Page.js is a client-side router inspired by Express routes to direct users to specific views.
