JavaScript Toolkit Utilities & Support

DOM Utilities

DOM Utilities are JavaScript libraries to help you come across difference on the API of different browsers, exposing an standard interface when manipulating the rendered elements. Some of those tools also provide extra utilities, like animations or data loading.

D3

D3 is a JavaScript visualization library for HTML and SVG, it lets you create things like graphs, charts, diagrams, maps and so on.

Links

  • http://d3js.org/

Handlebars

An extension to the Mustache templating language, Handlebars provides the power necessary to let you build semantic templates effectively with no frustration.

Links

  • http://handlebarsjs.com/

jQuery

jQuery is set of JavaScript utilities that helps the interaction with the DOM and other browser functionalities, it excels in browser compatibility as it support several browsers.

Links

  • http://jquery.com/

Mustache

Minimal templating with {{mustaches}} in JavaScript.

Links

  • https://github.com/janl/mustache.js

Polymer

A library that leverages the creation of web components, driving the future of the web through the usage of the newest standards and technologies.

Links

  • http://www.polymer-project.org/

Prototype

One of the first JavaScript utility libraries, it extends DOM and provide useful functionalities for the JavaScript developer. Along with jQuery, Prototype.js helped shape most of what we see in the JavaScript community nowadays.

Links

  • http://prototypejs.org/
  • https://github.com/sstephenson/prototype/

three.js

A lightweight 3D library with a very low level of complexity, it provides <canvas>, <svg>, CSS3D and WebGL renderers.

Links

  • http://threejs.org/

Zepto

Zepto is a library very similar to jQuery, although it's very small and focused on modern browsers only. It focus on having a small footprint in order to be faster on mobile devices as well.

Links

  • http://zeptojs.com/
