JavaScript Utilities

Libraries to transform data and handle its flow, with promises, streams or other async functions, using reactivity and functional programming approaches.

Async

Async is an utility library that provide asynchronous functions to control flow, to make it easier to deal with callback-oriented program execution.

Links

  • https://github.com/caolan/async

Bacon JS

A small functional library that enables declarative event handling.

Links

  • http://baconjs.github.io/

fn.js

fn.js is an opinionated JavaScript library that pushes you to to adopt strategies that encourage a functional programming style.

Links

  • http://eliperelman.com/fn.js/

Immutable

Immutable persistent data collections for Javascript which increase efficiency and simplicity.

Links

  • http://facebook.github.io/immutable-js/

Lazy.js

Lazy.js is a functional library similar to Underscore's API, with the computations lazily evaluated when the values are needed.

Links

  • http://danieltao.com/lazy.js/

Lo-Dash

A utility library delivering consistency, customization, performance, & extras. Lo-Dash gives you methods for dealing with collections in a more functional-style way, and several other utilities methods for functions, arrays, objects, etc.

Links

  • http://lodash.com/

Q js

A tool for creating and dealing with asynchronous promises in JavaScript. Q implements Promises/A+ standard.

Links

  • https://github.com/kriskowal/q

Ramda

A practical functional library designed specifically for a functional programming style, that makes it easy to create functional pipelines, and never mutates user data.

Links

  • http://ramdajs.com/

RxJS

The Reactive Extensions for JavaScript is a set of libraries to compose asynchronous and event-based programs using observable collections and arrays composition in JavaScript.

Links

  • https://github.com/Reactive-Extensions/RxJS

Underscore

Underscore provides over 100 functions that support both your favorite workaday functional helpers: map, filter, invoke — as well as more specialized goodies: function binding, JavaScript templating, creating quick indexes, deep equality testing, and so on.

Links

  • http://underscorejs.org/
